속성 선택자

요소[속성] : 특정 속성을 가진 객체 선택
요소[속성 = 값] : 속성 안의 값이 특정 값과 같은 객체 선택
요소[속성 ~= 값] : 속성 안의 값이 특정 값을 단어로써 포함하는 같은 객체 선택
요소[속성 ^= 값] : 속성 안의 값이 특정 값으로 시작하는 객체 선택
요소[속성 $= 값] : 속성 안의 값이 특정 값으로 끝나는 객체 선택
요소[속성 *= 값] : 속성 안의 값이 특정 값을 포함하는 객체 선택
